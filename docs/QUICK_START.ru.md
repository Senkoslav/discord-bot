# ⚡ Быстрый старт (5 минут)

## 📋 Что нужно сделать

1. ✅ Создать бота в Discord
2. ✅ Получить токен
3. ✅ Получить свой ID
4. ✅ Настроить .env
5. ✅ Запустить

---

## 1️⃣ Создай бота (2 минуты)

### Открой: https://discord.com/developers/applications

```
┌─────────────────────────────────────────┐
│  Discord Developer Portal               │
├─────────────────────────────────────────┤
│                                         │
│  [New Application]  ← Нажми сюда       │
│                                         │
└─────────────────────────────────────────┘
```

### Придумай имя → Create

```
┌─────────────────────────────────────────┐
│  Create an Application                  │
├─────────────────────────────────────────┤
│  Name: [Music Bot____________]          │
│                                         │
│  [Cancel]  [Create] ← Нажми            │
└─────────────────────────────────────────┘
```

## 2️⃣ Получи токен (1 минута)

### Слева выбери "Bot"

```
┌──────────────┬──────────────────────────┐
│ General      │                          │
│ OAuth2       │  TOKEN                   │
│ ► Bot        │  ┌────────────────────┐  │
│ Installation │  │ [Reset Token]      │  │
│              │  └────────────────────┘  │
└──────────────┴──────────────────────────┘
```

### Нажми "Reset Token" → "Yes, do it!"

```
┌─────────────────────────────────────────┐
│  Your token:                            │
│  ┌───────────────────────────────────┐  │
│  │ MTIzNDU2Nzg5MDEyMzQ1Njc4.GhJ... │  │
│  │                                   │  │
│  │ [Copy] ← СКОПИРУЙ И СОХРАНИ!     │  │
│  └───────────────────────────────────┘  │
│                                         │
│  ⚠️ Токен больше не покажется!          │
└─────────────────────────────────────────┘
```

### Прокрути вниз → Включи "Message Content Intent"

```
┌─────────────────────────────────────────┐
│  Privileged Gateway Intents             │
```

## 3️⃣ Пригласи бота (1 минута)

### Слева выбери "OAuth2" → "URL Generator"

```
┌──────────────┬──────────────────────────┐
│ General      │  SCOPES                  │
│ OAuth2       │  ☑ bot                   │
│   URL Gen... │  ☑ applications.commands │
│ Bot          │                          │
└──────────────┴──────────────────────────┘
```

### Выбери права:

```
┌─────────────────────────────────────────┐
│  BOT PERMISSIONS                        │
├─────────────────────────────────────────┤
│  Text Permissions:                      │
│  ☑ Send Messages                        │
│  ☑ Embed Links                          │
│                                         │
│  Voice Permissions:                     │
│  ☑ Connect                              │
│  ☑ Speak                                │
│  ☑ Use Voice Activity                   │
└─────────────────────────────────────────┘
```

### Скопируй ссылку внизу → Открой в браузере → Выбери сервер

---

## 4️⃣ Получи свой ID (30 секунд)

### В Discord: Настройки → Расширенные → Режим разработчика ВКЛ

```
┌─────────────────────────────────────────┐
│  Расширенные                            │
├─────────────────────────────────────────┤
│  Режим разработчика        [ВКЛ] ←     │
└─────────────────────────────────────────┘
```

### Правой кнопкой на своё имя → "Копировать ID пользователя"

```
     Твой профиль
     ┌─────────────────────┐
     │ @твоё_имя           │
     ├─────────────────────┤
     │ Профиль             │
     │ Копировать ID ←     │
     │ Упомянуть           │
     └─────────────────────┘
```

---

## 5️⃣ Настрой .env (1 минута)

```bash
cd music-bot
cp .env.example .env
```

### Открой .env и заполни:

```env
# Вставь токен из шага 2
DISCORD_TOKEN=MTIzNDU2Nzg5MDEyMzQ1Njc4.GhJ...

# Вставь ID из шага 4
BOT_OWNER_ID=123456789012345678

# Оставь как есть
USE_REDIS=false
```

---

## 6️⃣ Запусти! (3-5 минут первый раз)

```bash
docker compose up --build
```

### Увидишь:

```

✓ Container music-bot        Started

music-bot  | Bot is ready! Logged in as Music Bot
music-bot  | Connected to 1 guilds
```

## 🆘 Не работает?

### Бот не запускается
```bash
# Проверь, что Docker запущен:
docker --version

# Проверь .env файл:
cat .env
```

### Бот не отвечает
```bash
# Подожди 1-2 минуты после запуска
# Или перезапусти:
docker compose restart
```

### Нет звука
```bash
# Проверь логи:
docker compose logs bot

# Убедись что у бота есть права Connect и Speak
```
